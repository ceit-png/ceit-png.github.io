Certificate Course in Java Programming

The objective of this course is to provide students with expertise in Java Programming. This includes both the Core Java and Advanced Java programming. After doing the course, the student will be able to design, develop and maintain web-based enterprise applications effectively.
Duration : 16 - 18 Weeks (320 Hours) Course Fee : K 5,060
Eligibility : Minimum of Grade 12 Certificate     NSQF Level : 7
Course Prerequisite : Fair knowledge of Computing Fundamentals
